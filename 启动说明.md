# 知学在线教育平台 - 本地启动说明

> 本地开发环境启动顺序指南

---

## 第一步：启动中间件（命令行）

### Redis
```powershell
cd D:\redis-server
redis-server.exe
```
> Redis 默认端口：6379

---

### Nacos（单节点模式）
```powershell
cd D:\nacos\nacos\bin
./startup.cmd -m standalone
```
> Nacos 默认端口：8848
> 浏览器访问：http://localhost:8848/nacos
> 默认账号/密码：nacos / nacos

---

### MinIO
```powershell
cd D:\minio
.\minio.exe server D:\minio-data --console-address ":9001"
```
> MinIO API 端口：9000
> MinIO Console 端口：9001
> 浏览器访问：http://localhost:9001
> 默认账号/密码：minioadmin / minioadmin

---

### Sentinel Dashboard
```powershell
cd D:\tool\sentinel
start-sentinel.bat
```
> Sentinel 默认端口：8858
> 浏览器访问：http://localhost:8858

---

## 第二步：IDEA 中按顺序启动

## 第三步：启动前端

### 管理后台
```powershell
cd F:\online-edu\edu-admin && npm run dev
```
> 访问地址：http://localhost:3001

---

### 用户前台
```powershell
cd F:\online-edu\edu-web && npm run dev
```
> 访问地址：http://localhost:3000

## 常见问题

### Q1: 启动后端报数据库连接错误
> 确保 MySQL 服务已启动，检查 `application.yml` 中的数据库配置

### Q2: 启动后端报 Redis 连接错误
> 确保 Redis 已启动，端口 6379 可用

### Q3: 前端请求 404
> 确保后端微服务全部启动，网关路由正常

### Q4: 支付宝支付无法回调
> 确保 natapp 内网穿透运行中，notify-url 配置正确

### Q5: MinIO 上传失败
> 确保 MinIO 服务运行，Bucket `online-edu` 已创建并设置公开读
